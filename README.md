# CIFAR-10 Image Classification — CSCE 478

Image classification on CIFAR-10 comparing a logistic regression baseline against a CNN.

**Team:** Dat Bui, Amy Nguyen, Nick Pham

## How to Run

```bash
# 1. install dependencies
pip install -r requirements.txt

# 2. preprocess data + train baseline
python cifar10_preprocessing.py

# 3. train CNN + generate all figures
python cnn_model.py
```

Run them in order — `cnn_model.py` depends on the `.npy` files that `cifar10_preprocessing.py` saves.

## Scripts

| File | What it does |
|------|-------------|
| `cifar10_preprocessing.py` | Loads CIFAR-10, normalizes pixels, trains logistic regression baseline, saves `.npy` data |
| `cnn_model.py` | Builds and trains a 3-block CNN, evaluates it, generates all comparison plots |

## Results

| Model | Accuracy | Macro-F1 | 95% CI (Accuracy) |
|-------|----------|----------|-------------------|
| Logistic Regression | 39.51% | 0.3913 | [0.3853, 0.4046] |
| CNN | 69.51% | 0.6899 | [0.6857, 0.7042] |

## Output Files

### `DataFiles/`

Preprocessed data saved as `.npy` (NumPy binary). Generated by `cifar10_preprocessing.py`.

- `X_train.npy` — normalized training images (50000, 32, 32, 3)
- `X_test.npy` — normalized test images (10000, 32, 32, 3)
- `X_train_flat.npy` — flattened for logreg (50000, 3072)
- `X_test_flat.npy` — flattened for logreg (10000, 3072)
- `y_train.npy` — training labels (50000,)
- `y_test.npy` — test labels (10000,)

### `FigureFiles/`

All plots for the report. Generated by both scripts.

| File | Description |
|------|-------------|
| `cifar10_samples.png` | One sample image per class |
| `class_distribution.png` | Bar chart of class counts |
| `baseline_confusion_matrix.png` | Baseline normalized confusion matrix |
| `cnn_training_curve.png` | Train/val accuracy and loss over 20 epochs |
| `cnn_confusion_matrix.png` | CNN normalized confusion matrix |
| `baseline_vs_cnn_accuracy.png` | Per-class accuracy comparison |
| `misclassified_examples.png` | 20 images the CNN got wrong |
| `gradcam_heatmaps.png` | Grad-CAM attention maps per class |
| `tsne_features.png` | t-SNE of CNN penultimate layer features |
